AUTHOR

ABOUT

This is a resource type plugin that allows you to add
YouTube and Vimeo video's to your Moodle course.

It includes a JQuery javascript file that makes sure
the video is displayed as a responsive video

INSTALLTION

Install into the /mod folder

CONFIGURATION

Add a course activity
Select webvideo
Add your full video URL from Youtube or Vimeo
All done!
